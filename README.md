# 如何使用

1. 在项目的pom.xml中添加

```xml
    <repositories>
        <repository>
            <id>jitpack.io</id>
            <url>https://jitpack.io</url>
        </repository>
    </repositories>
```



```xml
    <dependency>
        <groupId>com.github.HaoLi-Rainbow</groupId>
        <artifactId>minio-spring-boot-starter</artifactId>
        <version>Tag</version>
    </dependency>
```
